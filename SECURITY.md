## Reporting a Vulnerability

We take the security of Home Server Pro seriously. If you discover a security vulnerability, please do not open a public GitHub issue. Instead, help us protect the community by reporting it responsibly.

### How to Report
Please report security vulnerabilities by creating a **Repository Security Advisory** directly through this GitHub repository's "Security" tab, or reach out to the project maintainer via the contact methods listed on the [LimestoneLMG GitHub Profile](https://github.com/LimestoneLMG).

Please include the following details in your report:
* A description of the vulnerability and its potential impact.
* Detailed steps to reproduce the issue (proof-of-concept scripts or screenshots are highly appreciated).
* The specific version and environment where the issue was found.

### Our Process
After receiving your report, we will:
1. Acknowledge receipt of the vulnerability report within 48 hours.
2. Work on a fix or mitigation plan.
3. Keep you updated on our progress and coordinate a public disclosure date once a patch is ready.
